Sejong Korean 1

Lesson 3: 물건 사기 (Buying Things)

Learning Objectives

  • Learn vocabulary for items in a store.
  • Ask for prices and availability of items.
  • Use Native Korean numbers with counters (like 개).
  • Connect nouns using 하고 (and).
  • Politely request items using 주세요.
