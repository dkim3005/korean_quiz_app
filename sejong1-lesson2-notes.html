Lesson 2: 일상생활 (Daily Life)

Learning Objectives

  • Describe daily activities like eating, sleeping, and working.
  • Use the formal polite verb ending -습니다/ㅂ니다.
  • Use the object marking particle 을/를.
  • Correctly use location particles 에 (to a place) and 에서 (in a place).

Core Grammar

1. V + -습니다/ㅂ니다 (Formal Ending)

This is a formal, polite verb ending used in formal situations like news, presentations, or the military.
• Use -습니다 after a verb stem ending in a consonant.
• Use -ㅂ니다 after a verb stem ending in a vowel.

가다 (to go) → 가 + ㅂ니다 → 갑니다.

먹다 (to eat) → 먹 + 습니다 → 먹습니다.

2. N + 을/를 (Object Particle)

This particle attaches to a noun to show that it is the direct object of the verb.
• Use 을 after a noun ending in a consonant.
• Use 를 after a noun ending in a vowel.

커피를 마십니다. (I drink coffee.)

밥을 먹습니다. (I eat a meal.)

3. N + 에 vs. 에서 (Location Particles)

These particles both relate to location, but have a crucial difference.

  • 에 (to/at): Used for destination (with verbs like 가다/오다) or the location of existence (with 있다/없다). It indicates a static point.
  • 에서 (in/at): Used to mark the location where an action takes place.

학교에 갑니다. (I go to school.)

도서관에서 공부합니다. (I study at the library.)
